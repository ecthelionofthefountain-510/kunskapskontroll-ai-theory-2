# Kunskapskontroll - AI Theory 2

Denna mapp samlar kursens inlämningar, kod och anteckningar i ett och samma sammanhang. Fokus ligger på uppgifter som kompletterar kursens teori med praktisk implementation.

Den mer detaljerade projektbeskrivningen for RAG-chatboten finns i den separata README i [tolkien-rag-chatbot/](tolkien-rag-chatbot/README.md).

---

## Snabb översikt

- [tolkien-rag-chatbot/](tolkien-rag-chatbot/) - RAG-baserad chattbot (huvudprojekt)
- Notebookar för kursuppgifter och experiment
- Inlämningsfiler och bilder

- [chapter_10_assignment_8.ipynb](chapter_10_assignment_8.ipynb) - notebook för kapitel 10

## Kort om innehåll

Materialet är uppdelat i:

- **Notebookar** för analys, experiment och förklaringar.
- **Kodfiler** för uppgifter som lämnas in som ren pythonkod.
- **Resurser** som bilder och data som behovs i rapporter/notebookar.

## Komma igång

Om du vill köra notebookarna lokalt:

1. Skapa och aktivera en virtuell miljö.
2. Installera beroenden från [requirements.txt](requirements.txt).
3. Starta Jupyter och öppna relevant notebook.
