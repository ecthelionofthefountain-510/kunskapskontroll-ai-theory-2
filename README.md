# Kunskapskontroll - AI Theory 2

Denna mapp samlar kursens inlamningar, kod och anteckningar i ett och samma sammanhang. Fokus ligger pa uppgifter som kompletterar kursens teori med praktisk implementation.

Den mer detaljerade projektbeskrivningen for RAG-chatboten finns i den separata README i [tolkien-rag-chatbot/](tolkien-rag-chatbot/README.md).

---

## Snabb oversikt

- [tolkien-rag-chatbot/](tolkien-rag-chatbot/) - RAG-baserad chattbot (huvudprojekt)
- Notebookar for kursuppgifter och experiment
- Inlamningsfiler och bilder

---

## Mappstruktur (kort)

- [chapter_10_assignment_8.ipynb](chapter_10_assignment_8.ipynb) - notebook for kapitel 10
- [chapter_8_assignment_9b.py](chapter_8_assignment_9b.py) - pythonkod for kapitel 8
- [kunskapskontroll_kod.ipynb](kunskapskontroll_kod.ipynb) - samlad kod for kursmoment
- [images/](images/) - figurer och skarmbilder
- [tolkien-rag-chatbot/](tolkien-rag-chatbot/) - separat projekt med egen README

## Kort om innehall

Materialet ar uppdelat i:

- **Notebookar** for analys, experiment och forklaringar.
- **Kodfiler** for uppgifter som lamnas in som ren pythonkod.
- **Resurser** som bilder och data som behovs i rapporter/notebookar.

---

## Komma igang

Om du vill kora notebookarna lokalt:

1. Skapa och aktivera en virtuell miljo.
2. Installera beroenden fran [requirements.txt](requirements.txt).
3. Starta Jupyter och oppna relevant notebook.
